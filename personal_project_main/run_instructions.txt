HOW TO RUN THE MESSAGE LOG PROJECT
====================================

1. Start the Flask Server
   In your terminal, run:
   
   cd /Users/turtlesoup/Repos/se3200_web_app/personal_project_main
   python3 message_log_server.py
   
   The server will start on http://localhost:5000

2. Open the HTML File in Your Browser
   
   Option A - From Terminal:
   open /Users/turtlesoup/Repos/se3200_web_app/personal_project_main/message_log.html
   
   Option B - From Browser:
   - Press Cmd + O (File > Open File)
   - Navigate to: /Users/turtlesoup/Repos/se3200_web_app/personal_project_main/message_log.html
   - Click Open
   
   Option C - From Finder:
   - Navigate to /Users/turtlesoup/Repos/se3200_web_app/personal_project_main/
   - Double-click message_log.html

3. The application will load and automatically fetch messages from the server.
   You can post new messages using the form, and they'll be saved to messages.txt

TESTING WITH CURL
=================

Test GET request:
curl http://localhost:5000/messages

Test POST request:
curl -X POST http://localhost:5000/messages \
  -H "Content-Type: application/json" \
  -d '{"message": "Test message from curl"}'

Test 404 error:
curl http://localhost:5000/invalid-route
